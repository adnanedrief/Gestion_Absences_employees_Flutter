# Gestion_Absences_employees_Flutter
Une application mobile sous Flutter de gestion d'absence des employées avec dépot de justification et un système de marquage d'entrée et de sortie

# Installation

```javascript
flutter clean
flutter pub get
```

# Utilisation


```javascript
flutter run
```
